## 🎯 リポジトリの目的

本書で解説した分析手法を、各言語でどのように記述するか、その具体的な実装例を提示することを目的としています。

数式や理論が実際のスクリプトとしてどのように構成されるのかを確認するための「実装リファレンス」としてご活用ください。

読者の皆様がご自身の環境で分析を組み立てる際の、実務的なガイドとなることを目指しています。

---

## 📂 リポジトリ構成

本リポジトリは、言語別にフォルダを分けて構成しています。

- **data/**  
  分析に使用する共通サンプルデータ（CSV、Excel、dta形式など）

- **python/**  
  Python（主に Jupyter Notebook 形式）によるコード例

- **r/**  
  R（.Rファイル）によるコード例

- **stata/**  
  Stata（.doファイル）によるコード例

各プログラムは、ルート直下の `data/` フォルダ内のデータを参照する構成となっています。
